# mac系统 + Cursor 快捷键整理（Markdown版）

说明：⌘=Command，⌥=Option，⌃=Control，所有快捷键适配macOS系统，Cursor快捷键为mac版专属。

# 一、macOS 系统常用快捷键

## 1. 通用操作（全系统适用）

|快捷键|功能|
|---|---|
|⌘ + C|复制|
|⌘ + V|粘贴|
|⌘ + X|剪切|
|⌘ + Z|撤销|
|⌘ + Shift + Z|重做|
|⌘ + A|全选|
|⌘ + F|查找|
|⌘ + S|保存|
|⌘ + P|打印|
|⌘ + N|新建窗口/文件|
|⌘ + O|打开文件|
|⌘ + W|关闭当前窗口|
|⌘ + Q|退出当前应用|
|⌘ + Tab|切换应用|
|⌘ + `|切换同应用多窗口|
|⌘ + H|隐藏当前应用|
|⌘ + ⌥ + H|隐藏其他应用|
|⌘ + M|最小化窗口|
|⌘ + ⌥ + M|最小化所有窗口|
|⌃ + ⌘ + Q|快速锁屏|
|⌘ + ⌥ + Esc|强制退出应用|
## 2. 访达（Finder）文件管理

|快捷键|功能|
|---|---|
|⌘ + Shift + N|新建文件夹|
|⌘ + Delete|移到废纸篓|
|⌘ + Shift + Delete|清空废纸篓|
|⌘ + ↑|进入上一级文件夹|
|⌘ + ↓|进入下一级/打开文件|
|⌘ + [ / ]|前进/后退|
|⌘ + I|显示简介|
|⌘ + Shift + .|显示/隐藏隐藏文件|
## 3. 窗口与桌面

|快捷键|功能|
|---|---|
|⌃ + ⌘ + F|进入/退出全屏|
|⌃ + ↑|调度中心|
|⌃ + ↓|应用窗口|
|⌃ + ←/→|切换桌面|
|⌘ + ⌥ + D|显示/隐藏程序坞|
## 4. 文本编辑（通用）

|快捷键|功能|
|---|---|
|⌘ + ←/→|光标到行首/行尾|
|⌥ + ←/→|光标按单词移动|
|⌘ + ↑/↓|光标到文档开头/结尾|
|Shift + 方向键|选中文字|
|fn + Delete|向前删除（相当于Win的Delete）|
# 二、Cursor 编辑器快捷键（macOS版）

## 1. 基础编辑

|快捷键|功能|
|---|---|
|⌘ + S|保存文件|
|⌘ + ⌥ + S|保存所有文件|
|⌘ + Z|撤销|
|⌘ + Shift + Z|重做|
|⌘ + X|剪切行（无选中时）|
|⌘ + C|复制行（无选中时）|
|⌘ + /|注释/取消注释|
|⌘ + ] / [|缩进/取消缩进|
|Shift + ⌘ + K|删除行|
|⌥ + ↑/↓|移动行|
|Shift + ⌥ + ↑/↓|复制行|
## 2. 光标与选择

|快捷键|功能|
|---|---|
|⌘ + ←/→|行首/行尾|
|⌥ + ←/→|按单词移动|
|⌘ + ↑/↓|文件开头/结尾|
|⌘ + D|选中下一个相同文本|
|⌘ + Shift + L|选中所有相同文本|
|⌥ + 点击|多光标编辑|
## 3. 文件与导航

|快捷键|功能|
|---|---|
|⌘ + P|快速打开文件|
|⌘ + Shift + P|打开命令面板|
|⌘ + B|切换侧边栏|
|⌘ + \|分屏|
|⌃ + `|打开/关闭终端|
|⌘ + F|查找|
|⌘ + Shift + F|全局查找|
## 4. AI 辅助（Cursor 特色）

|快捷键|功能|
|---|---|
|⌘ + K|打开内联 AI 编辑|
|⌘ + Shift + K|添加选中到 AI 编辑|
|⌘ + L|新建 AI 聊天|
|⌘ + Shift + L|添加选中到聊天|
|Tab|接受 AI 代码建议|
|⌘ + Enter|提交 AI 编辑|
> （注：文档部分内容可能由 AI 生成）
